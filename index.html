Welcome to the GitHub Page of netTransformer

netTransformer is a conceptual software prototype of a network transformation manager able to:

  • Discover current network infrastructure
  • Capture its state
  • Build a graph data model out of it
  • Provide ability to review and reason about L2, L3, OSPF, ISIS and BGP topology
  • Automate device configuration process through simplified template interface!
  • Track the network evolution process
  • Transform a network from one state to the other

Authors and Contributors

In 2010, Nikolay Milovanov (@niau) and Vasil Yordanov (@vasil-yordanov) created the first prototype of netTransformer. netTransformer initially has been developed as part of the Phd of Nikolay in "IPv4 to IPv6 network transformation" and has been used for discovery and formalization in a data model of the current network state, modeling of the IPv4 to IPv6 network transformation strategy steps and their execution. Nikolay was also the person to develop most of the discovery methods and the integration with the network. Vasil always was the primary development force behind the project. To Vasil we also own most of the low level design, the split into multiple components and most of the code base.
